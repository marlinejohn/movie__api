MYFLIX APP

Objective

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

ENDPOINTS AND URLs

REQUEST URL HTTP METHOD REQUEST BODY DATA FORMAT RESPONSE BODY DATA FORMAT
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all movies
Return data about a single movie by title to the user /movies/[title] GET None A JSON object holding data about a single movie, id, title, genre, director. Example: { id: 10, title: 'Twilight', director: 'Catherine Hardwicke', genre: 'Suspence Romance' }
Return data about a genre by ID /movies/id/[idNumber] GET None A JSON object holding data about a single movie, id, title, genre, director. Example: { id: 10, title: 'Twilight', director: 'Catherine Hardwicke', genre: 'Suspence Romance' }
Return data about a genre by name /movies/genre/[genreName] GET None A JSON object holding data about a single movie, title, genre, director. Example: { id: 10, title: 'Twilight', director: 'Catherine Hardwicke', genre: 'Suspence Romance' }
Return data about a genre by director name /movies/directors/[directorName] GET None A JSON object holding data about a single movie, title, genre, director. Example: { id: 10, title: 'Twilight', director: 'Catherine Hardwicke', genre: 'Suspence Romance' }
Add a movie /movies POST A JSON object holding data about the movie to add, structured like:
{ title: 'Twilight',
director: 'Catherine Hardwicke',
genre: 'Suspence Romance' } 		A JSON object holding data about the movie that was added, including an ID:
{ id : ###,
title: 'Twilight',
director: 'Catherine Hardwicke',
genre: 'Suspence Romance' }
Update a movie from the movies list /movies/[idNumber] PUT None A text message indicating whether the movie was updated.
Remove a movie from the movies list /movies/[idNumber] DELETE None A text message indicating whether the movie was successfully removed.
